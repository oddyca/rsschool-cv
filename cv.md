## Code example

> Codewars 6kyu kata

**Description**: Complete the solution so that the function will break up camel casing, using a space between words.

**Solution**:
```
function solution(string) {
  let regEx = /[a-z]+[A-Z]/
    if (regEx.test(string)) {
      let newStr = string.replace(/([a-z]+)([A-Z])/g, '$1 $2')
      return newStr
    } else {
      return string
    }
}
```
